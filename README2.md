# HTTP Header Inspector

A Python script that fetches and displays HTTP response headers.

## Description
This project helps inspect server response headers for websites or APIs.

## Features
- Fetch headers from URL
- Useful for debugging
- Beginner-friendly networking tool

## Technologies Used
- Python
- urllib module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python header_inspector.py

4. Enter URL

## Example

Input:
https://example.com

Output:
Content-Type: text/html  
Server: nginx
